Produkty — checklisty

Odhaczaj warunki na bieżąco. Stan zapisuje się w tej przeglądarce — włącz „☁️ Synchronizacja", by mieć go też na innych urządzeniach.

☁️ Synchronizacja przez GitHub Gist

Stan odhaczeń trzymany w prywatnym Gist na Twoim koncie GitHub — to samo na telefonie i na kompie. Token wklejasz raz na każdym urządzeniu; zostaje tylko w tej przeglądarce, nie trafia do repo.

—

Jak zdobyć token?

GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic) → Generate new token → zaznacz tylko zakres gist → skopiuj i wklej powyżej. (Token „classic" jest najprostszy; fine-grained z uprawnieniem „Gists: Read and write" też zadziała.)
